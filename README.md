# 🥗 Macros Calculator

A simple and interactive web-based **macronutrient calculator** that helps users estimate their daily calorie needs and macronutrient breakdown based on personal data and fitness goals.

🔗 **Live Demo:** https://oleksandrk-k-kodehode.github.io/macros-calculator/

---

## 📌 Overview

This application allows users to input basic information such as age, weight, height, gender, and activity level to calculate:

- Daily calorie needs
- Macronutrient distribution (Protein, Carbohydrates, Fat)
- Personalized nutrition targets

Macro calculators are commonly used in fitness and nutrition planning to help users reach goals like **weight loss, muscle gain, or maintenance** ([GitHub][1]).

---

## ⚙️ Features

- 🧮 **Calorie Calculation** – Estimates daily caloric needs
- 🍗 **Macro Breakdown** – Calculates protein, carbs, and fats
- 📊 **User-Friendly Interface** – Simple and intuitive UI
- 🎯 **Goal-Based Results** – Supports different fitness goals
- 📱 **Responsive Design** – Works on desktop and mobile

---

## 🧠 How It Works

1. User enters personal data:
    - Age
    - Height
    - Weight
    - Gender
    - Activity level

2. The app calculates:
    - **BMR (Basal Metabolic Rate)**
    - **TDEE (Total Daily Energy Expenditure)**

3. Based on the selected goal, it distributes calories into:
    - Protein
    - Carbohydrates
    - Fats

These values help users plan their daily nutrition intake effectively.
